A Starter Project for RequireJS/jQuery/SVG

This is a project utilizing RequireJS/jQuery/SVG (and some Bootstrap for front-end prettiness). It can be easily used as a starter project using these tools. This project is also used as a running collection of items focusing on these tools.

1. Clickable statically created SVG Elements

Statically created SVG elements can be accessed by ID and can respond to click events fairly simply:

<-- Click this rect!

2. Clickable dynamically-crated SVG via jQuery

Typically, SVG objects are not accessible via jQuery. This is because SVG elements cannot be parsed by innerHTML. bobince from Stack Overflow provides a wonderful explanation, including JavaScript logic to create SVG elements using document.createElementNS.
